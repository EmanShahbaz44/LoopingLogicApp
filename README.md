# Looping Logic App

A Windows Forms application built in Visual Basic to showcase how loop structures work in programming. 

## About The Project
This project was created to demonstrate the practical use of `For` and `Do While` loops. The application features a graphical user interface (GUI) with buttons that trigger different looping behaviors and display the output in a text box.

## Features
* **Punishment Writer:** Uses a `Do While` loop to automatically generate and number a sentence 25 times ("I will complete my assignments on time.").
* **99 Bottles of Root Beer:** Uses both `Do While` and `For` loops to correctly count down and print the lyrics to the classic song.
* Demonstrates string formatting and appending text dynamically to a RichTextBox.

## Built With
* Visual Basic .NET
* Windows Forms
